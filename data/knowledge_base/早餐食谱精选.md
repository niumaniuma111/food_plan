# 早餐食谱精选

## 小米粥

### 食材
- 小米 100克
- 清水 800毫升
- 红枣 3颗（可选）
- 枸杞 少许（可选）

### 做法
1. 小米淘洗干净，红枣去核切半
2. 锅中加水烧开，放入小米和红枣
3. 大火烧开后转小火，熬煮30分钟
4. 期间搅拌几次，防止粘锅
5. 出锅前5分钟加入枸杞
6. 煮至粥浓稠、米油浮起即可

### 小贴士
- 水开后再下小米，更容易煮出米油
- 小米粥不宜加碱，会破坏营养成分
- 搭配鸡蛋和馒头，是一顿营养均衡的早餐

## 鸡蛋饼

### 食材
- 鸡蛋 3个
- 面粉 100克
- 清水 150毫升
- 葱花 适量
- 盐 适量
- 食用油 适量

### 做法
1. 面粉加清水搅拌成无颗粒的稀面糊
2. 鸡蛋打入面糊中，加盐搅匀
3. 加入葱花拌匀
4. 平底锅刷一层薄油，小火加热
5. 倒入一勺面糊，转动锅让面糊摊平
6. 煎至底部金黄，翻面再煎30秒
7. 取出切块即可食用

### 小贴士
- 面糊不要太稠，否则饼会太厚
- 小火慢煎，避免外焦里生
- 可以加入火腿丁、胡萝卜丝等增加口感

## 燕麦牛奶

### 食材
- 即食燕麦片 50克
- 纯牛奶 250毫升
- 蜂蜜 1汤匙（可选）
- 坚果碎 适量（可选）
- 蓝莓或香蕉片 适量（可选）

### 做法
1. 将牛奶倒入锅中加热至微沸
2. 加入燕麦片，搅拌均匀
3. 小火煮2-3分钟至燕麦变软
4. 倒入碗中，根据喜好加入蜂蜜、坚果、水果

### 小贴士
- 牛奶不宜煮沸太久，会损失营养
- 燕麦煮的时间越长越软糯，可根据喜好调整
- 加入坚果和水果可以增加膳食纤维和维生素

## 豆浆配油条

### 食材
- 黄豆 100克
- 清水 800毫升
- 白糖 适量

### 做法
1. 黄豆提前浸泡8小时（或过夜）
2. 泡好的黄豆加清水放入破壁机/豆浆机
3. 选择豆浆模式，打成豆浆
4. 过滤豆渣，加白糖调味
5. 油条可购买半成品，烤箱或空气炸锅加热即可

### 小贴士
- 黄豆一定要泡透，否则豆浆口感粗糙
- 自制豆浆比市售更浓醇，营养更丰富
- 豆浆必须彻底煮沸，未煮熟的豆浆含有有害物质
