# 食物搭配与饮食禁忌

## 食物搭配宜忌

### 相克搭配（避免同食）
- **柿子 + 螃蟹**: 易引起腹痛、腹泻，两者都性寒，同食伤胃
- **菠菜 + 豆腐**: 菠菜中的草酸与豆腐中的钙结合形成草酸钙，影响钙吸收
- **牛奶 + 橙子**: 牛奶中的蛋白质与橙子中的果酸反应，影响消化吸收
- **海鲜 + 啤酒**: 同食会增加尿酸，容易引发痛风
- **鸡蛋 + 豆浆（未煮熟）**: 生豆浆中的胰蛋白酶抑制剂影响蛋白质消化

### 相宜搭配（推荐同食）
- **番茄 + 鸡蛋**: 番茄的维生素C促进鸡蛋中铁的吸收，营养互补
- **排骨 + 玉米**: 排骨补钙，玉米补膳食纤维，搭配营养均衡
- **西兰花 + 胡萝卜**: 两种蔬菜搭配，维生素C和β-胡萝卜素互补，增强免疫力
- **鱼肉 + 豆腐**: 鱼肉富含维生素D，促进豆腐中钙的吸收
- **瘦肉 + 青椒**: 瘦肉补铁，青椒富含维生素C促进铁吸收

## 特殊人群饮食注意

### 孕妇
- 避免生鱼片、生肉等生食，防止寄生虫感染
- 少吃高汞鱼类（如金枪鱼、旗鱼）
- 避免过量咖啡因，每天不超过200毫克
- 忌食薏米、马齿苋等可能引起子宫收缩的食物
- 多补充叶酸（深绿色蔬菜、豆类）

### 糖尿病患者
- 控制主食量，优先选择粗粮（糙米、燕麦）
- 避免高糖食物（甜点、含糖饮料）
- 少食多餐，稳定血糖
- 多吃蔬菜，增加膳食纤维摄入
- 注意水果的含糖量，选择低糖水果（如草莓、柚子）

### 高血压患者
- 严格控制钠盐摄入，每天不超过5克
- 少吃腌制食品、加工肉类
- 多吃富含钾的食物（香蕉、土豆、菠菜）
- 限制饮酒，最好不饮酒
- 增加富含钙和镁的食物（牛奶、豆制品）

### 痛风患者
- 避免高嘌呤食物（动物内脏、海鲜、浓肉汤）
- 限制饮酒，尤其是啤酒
- 多喝水，每天至少2000毫升
- 少吃高果糖食物（含糖饮料、蜂蜜）
- 适量摄入低脂乳制品，有助于降低尿酸

## 四季饮食禁忌

### 春季
- 少吃辛辣油腻，避免肝火旺盛
- 不宜吃太多热性食物（羊肉、辣椒）
- 避免食用发霉变质的食物（春季潮湿易霉变）

### 夏季
- 少吃冰镇食物，以免损伤脾胃
- 避免隔夜饭菜，夏季细菌繁殖快
- 不宜过量食用冷饮，影响消化功能

### 秋季
- 少吃辛辣烧烤，秋季干燥容易上火
- 避免过多食用生冷食物
- 不宜暴饮暴食，秋季进补要循序渐进

### 冬季
- 避免过度进补，荤素搭配要均衡
- 少吃寒凉食物，保护脾胃
- 不宜空腹吃柿子，容易形成胃石
