# 营养汤品食谱

## 番茄蛋花汤

### 食材
- 番茄 1个
- 鸡蛋 2个
- 葱花 适量
- 盐 适量
- 香油 少许
- 淀粉 1汤匙

### 做法
1. 番茄洗净切小块，鸡蛋打散备用
2. 锅中加少许油，放入番茄翻炒至出汁
3. 加入适量清水，大火烧开
4. 淀粉加水调成水淀粉，倒入汤中搅匀
5. 转小火，将蛋液沿筷子缓缓倒入，形成蛋花
6. 加盐调味，淋上香油，撒上葱花即可

### 小贴士
- 蛋液倒入后不要立刻搅拌，等几秒让蛋花成型
- 番茄先炒一下再煮汤，汤色更红更浓郁

## 紫菜蛋汤

### 食材
- 紫菜（干） 10克
- 鸡蛋 2个
- 虾皮 1汤匙
- 葱花 适量
- 盐 适量
- 香油 少许

### 做法
1. 紫菜撕成小片，鸡蛋打散
2. 锅中加水烧开，放入紫菜和虾皮煮2分钟
3. 加盐调味
4. 将蛋液缓缓倒入，形成蛋花
5. 淋上香油，撒上葱花即可

### 小贴士
- 虾皮本身有咸味，加盐时注意用量
- 紫菜不宜煮太久，否则口感会变差

## 排骨玉米汤

### 食材
- 猪排骨 400克
- 玉米 2根
- 胡萝卜 1根
- 姜 3片
- 料酒 1汤匙
- 盐 适量

### 做法
1. 排骨冷水下锅，加料酒焯水，去除血沫后捞出洗净
2. 玉米切段，胡萝卜切滚刀块
3. 将排骨、玉米、胡萝卜、姜片放入汤锅
4. 加入足量清水，大火烧开
5. 转小火慢炖1.5小时
6. 加盐调味即可

### 小贴士
- 水要一次加足，中途加水会影响汤的味道
- 小火慢炖能让排骨的营养充分释放到汤中
- 胡萝卜不宜过早放入，以免煮烂

## 冬瓜排骨汤

### 食材
- 猪排骨 300克
- 冬瓜 500克
- 姜 3片
- 葱 1根
- 料酒 1汤匙
- 盐 适量
- 枸杞 少许

### 做法
1. 排骨冷水下锅，加料酒焯水去血沫，捞出洗净
2. 冬瓜去皮去籽，切成大块
3. 将排骨、姜片放入汤锅，加足量清水
4. 大火烧开后转小火炖40分钟
5. 加入冬瓜块，继续炖20分钟
6. 加盐调味，撒上枸杞和葱花即可

### 小贴士
- 冬瓜性凉，夏天喝有清热解暑的功效
- 冬瓜不要炖太久，保持形状口感更好
